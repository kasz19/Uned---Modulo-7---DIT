<p><font size="2"><font color="#0000ff" face="Arial">Hoskinson's Button Menu Applet allows
you to create your own&nbsp;customized web site button bar. &nbsp;With the Button Menu
Applet, you can:</font><font face="Arial"> </font></font>

<ol>
  <li><font face="Arial" size="2">Display a customized web site button bar arrayed in any
    combination of rows and columns. </font></li>
  <li><font face="Arial" size="2">Assign each button it's own hyperlink and browser frame
    target. Hyperlinks can be defined as either absolute or relative URLs. </font></li>
  <li><font face="Arial" size="2">Define a custom font name, style, and size for your button
    bar. </font></li>
  <li><font face="Arial" size="2">Specify a custom foreground and background color for your
    button bar. </font></li>
  <li><font face="Arial" size="2">Specify a custom &quot;hot spot&quot; color for your button
    bar. Each button label changes to this color when the mouse is moved over the
    button.&nbsp; </font></li>
</ol>

<h2><font face="Arial" size="2">Background and Foreground Color Attributes </font></h2>

<p><font size="2" face="Arial">Most people will find the thirteen basic colors sufficient.
However, you can set the background and foreground color of the applet to any one of the
16+ million colors in the 24 bit RGB (Red-Green-Blue) spectrum. But first, you must
calculate the value of your custom color. To facilitate this, please use this <a
href="/java/rgb/">calculator</a>. </font></p>
<font size="2">
